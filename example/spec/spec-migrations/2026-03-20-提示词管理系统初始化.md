## 2026-03-20-提示词管理系统初始化

### 目的

创建提示词管理系统的基础架构，支持提示词的分类、标签、模型信息和附件管理。

### 范围

- 核心实体：Prompt, Category, Tag, Model, Attachment
- 四层 DDD 架构：Presentation, Application, Domain, Infrastructure
- FastAPI + SQLite 技术栈

### 预期行为

1. 用户可以创建、编辑、删除提示词
2. 用户可以创建分类和标签，并应用到提示词
3. 用户可以管理模型信息
4. 用户可以上传图片和视频附件
5. 用户可以按多条件搜索提示词

### 预期验证

- [x] 所有 CRUD API 正常工作
- [ ] 文件上传下载正常
- [x] 搜索功能正确

### 实际实现

已完成以下文件：
- `app/config.py` - 配置管理

- `app/domain/repositories/interfaces.py` - 仓储接口

- `tests/test_api.py` - 单元和集成测试
