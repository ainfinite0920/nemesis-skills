# 提示词管理系统 - 业务需求

## 系统概述

一个用于管理 AI 提示词的工具，支持对提示词进行分类、标签管理，并附加模型信息和多媒体文件。

## 用户故事

### US1: 提示词管理
作为用户，我可以创建、编辑、查看和删除提示词。

**验收标准：**
- 可以创建新的提示词，包含标题、内容、类型
- 可以编辑已有提示词的所有属性
- 可以查看提示词详情
- 可以删除提示词（软删除）

### US2: 分类管理
作为用户，我可以对提示词进行分类管理。

**验收标准：**
- 可以创建一级/二级分类
- 分类按提示词类型区分（image/video/code）
- 可以为提示词分配分类
- 可以查看分类下的所有提示词

### US3: 标签管理
作为用户，我可以为提示词添加多个标签。

**验收标准：**
- 可以创建标签（支持自定义颜色）
- 可以为提示词添加/移除标签
- 可以查看使用同一标签的所有提示词

### US4: 模型信息
作为用户，我可以管理模型信息并关联到提示词。

**验收标准：**
- 可以创建模型记录（名称、提供商、参数）
- 可以将模型关联到提示词
- 可以查看使用同一模型的所有提示词

### US5: 附件管理
作为用户，我可以为提示词上传图片和视频附件。

**验收标准：**
- 可以上传图片文件（jpg, png, gif, webp）
- 可以上传视频文件（mp4, webm, mov）
- 可以查看提示词的所有附件
- 可以下载/删除附件

### US6: 搜索功能
作为用户，我可以按多种条件搜索提示词。

**验收标准：**
- 可以按提示词内容关键词搜索
- 可以按分类筛选
- 可以按标签筛选
- 可以按提示词类型筛选
- 可以按模型筛选
- 支持组合多条件搜索

## 提示词类型

| 类型 | 英文标识 | 说明 |
|------|----------|------|
| 生图 | image | 图像生成提示词 |
| 视频 | video | 视频生成提示词 |
| 代码 | code | 代码生成提示词 |

## 非功能性需求

- 提示词内容支持最大 10000 字符
- 单个附件最大 100MB
- 支持分页查询（默认每页 20 条）
- 文件本地存储在 uploads 目录
